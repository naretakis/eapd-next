Testing Milkdown Demo Raw Markdown Functionality

This is a simple test to verify the demo page functionality.

Expected Behavior:

  1. Visit http://localhost:3000/milkdown-demo/
  2. Type some content in any of the three editors
  3. Click "Show Raw Markdown" button
  4. The raw markdown panels should appear on the right
  5. Click "Refresh Content" buttons to update the raw markdown display
  6. The raw markdown should show the current content from each editor

Key Changes Made:

  • Exposed getMarkdown() method via ref
  • Added refs to each editor instance in demo page
  • Updated raw markdown display to use editor refs instead of state
  • Added periodic updates and manual refresh buttons
